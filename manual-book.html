Manual Book WebGIS UAS

Dokumen ringkas ini menjelaskan struktur project, daftar aplikasi, dan konfigurasi environment database untuk deployment Docker atau Coolify.

Struktur Folder

Struktur root project disiapkan seperti berikut:

  • index.html sebagai landing page utama.
  • manual-book.html sebagai dokumentasi singkat.
  • 01 sampai 05 sebagai folder aplikasi WebGIS.

Environment Database

Setiap aplikasi PHP memakai environment variable berikut untuk koneksi database:

  • DB_HOST
  • DB_USER
  • DB_PASSWORD
  • DB_NAME

Jika database dipisah sebagai resource Coolify, isi DB_HOST dengan hostname service database, bukan localhost.
